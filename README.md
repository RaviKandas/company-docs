## Local Development

1. Install MkDocs:
   ```bash
   pip install mkdocs-material
   ```

2. Run locally:
   ```bash
   mkdocs serve
   ```

3. View at: http://localhost:8000

## Making Changes

1. Create a new branch
2. Make your changes in Markdown
3. Submit a Pull Request
4. Changes auto-deploy after merge
